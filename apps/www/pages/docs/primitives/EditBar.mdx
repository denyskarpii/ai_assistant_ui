---
title: EditBar
description: Buttons to manage the edit state.
---

## Anatomy

```tsx
import { EditBar } from "assistant-ui";

export default () => (
  <EditBar.Root>
    <EditBar.Save />
    <EditBar.Cancel />
  </ActionBar.Root>
);
```

## API Reference

### Root

Containts all parts of the EditBar.

### Save

Saves the changes.

### Cancel

Discards the changes.
